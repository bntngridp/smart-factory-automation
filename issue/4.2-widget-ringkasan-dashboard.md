# 4.2 Widget Ringkasan Dashboard

**Epic:** 4 — Dashboard Utama & Layout UI  
**Priority:** High  
**Dependencies:** 4.1, 1.1, 3.2

---

## Deskripsi

Membuat tampilan halaman utama (`/`) yang merangkum aktivitas pabrik secara real-time dalam bentuk kartu informasi.

## Acceptance Criteria

- [ ] Halaman `/` (dashboard) menampilkan minimal **3 kartu informasi**:
  1. **Total Jenis Produk** — jumlah seluruh produk di database.
  2. **Total Produksi Hari Ini** — total quantity dari `ProductionLogs` untuk tanggal hari ini.
  3. **Aktivitas Inventaris Terakhir** — 5 record terbaru dari `InventoryMovements` (menampilkan nama produk, quantity, tipe, dan timestamp).
- [ ] Data diambil langsung dari database via Server Component (tanpa API endpoint).
- [ ] Kartu menggunakan Tailwind CSS dengan warna dan ikon yang konsisten.
- [ ] Loading state untuk masing-masing kartu (Suspense boundary per kartu).
- [ ] Dashboard responsif — kartu grid 1 kolom di mobile, 3 kolom di desktop.

## Technical Notes

- Buat fungsi helpers di service layer:
  ```ts
  export async function getDashboardStats() {
    const totalProducts = await prisma.product.count()
    const todayProduction = await prisma.productionLog.aggregate({
      _sum: { quantity: true },
      where: { createdAt: { gte: startOfToday() } },
    })
    const recentMovements = await prisma.inventoryMovement.findMany({
      take: 5,
      orderBy: { createdAt: "desc" },
      include: { product: { select: { name: true } } },
    })
    return { totalProducts, todayProduction: todayProduction._sum.quantity ?? 0, recentMovements }
  }
  ```
- Struktur kartu:
  ```tsx
  <div className="bg-white rounded-lg shadow p-6">
    <div className="flex items-center gap-4">
      <div className="p-3 bg-blue-100 rounded-full">
        <PackageIcon className="w-6 h-6 text-blue-600" />
      </div>
      <div>
        <p className="text-sm text-gray-500">Total Jenis Produk</p>
        <p className="text-2xl font-bold">{totalProducts}</p>
      </div>
    </div>
  </div>
  ```

## Definition of Done

- Dashboard `/` menampilkan data akurat dari database.
- Kartu informasi diperbarui setiap kali halaman di-refresh (tidak ada cache stale).
- Loading state muncul saat data belum siap.
- Tampilan dashboard profesional dan informatif.

# 2.2 Antarmuka Tabel Master Produk

**Epic:** 2 — Modul Manajemen Produk (Master Data)  
**Priority:** High  
**Dependencies:** 2.1

---

## Deskripsi

Membangun halaman antarmuka menggunakan Tailwind CSS untuk menampilkan tabel data produk.

## Acceptance Criteria

- [ ] Halaman `/products` eksis dan dapat diakses dari browser.
- [ ] Tabel menampilkan kolom:
  - **ID** — kode unik produk.
  - **Nama Produk** — nama barang.
  - **Unit** — satuan (pcs, kg, liter, dll).
  - **Stok Minimal** — batas minimum stok.
- [ ] Status loading ditampilkan saat data sedang diambil.
- [ ] Gunakan komponen `Suspense` dengan fallback skeleton/loading spinner.
- [ ] Data difetch menggunakan `getProducts()` dari service layer.
- [ ] Tabel responsif (horizontal scroll pada layar kecil).
- [ ] Setiap baris memiliki aksi untuk mengedit stok minimal (modal atau inline).

## Technical Notes

- Layout dengan `page.tsx` + `loading.tsx` di folder `app/products/`.
- Contoh struktur tabel:

```tsx
export default async function ProductsPage() {
  const products = await getProducts()
  return (
    <div className="p-6">
      <h1 className="text-2xl font-bold mb-4">Master Produk</h1>
      <table className="w-full border-collapse">
        <thead>
          <tr className="bg-gray-100">
            <th className="p-2 text-left">ID</th>
            <th className="p-2 text-left">Nama Produk</th>
            <th className="p-2 text-left">Unit</th>
            <th className="p-2 text-left">Stok Minimal</th>
          </tr>
        </thead>
        <tbody>
          {products.map((p) => (
            <tr key={p.id} className="border-b">
              <td className="p-2">{p.id}</td>
              <td className="p-2">{p.name}</td>
              <td className="p-2">{p.unit}</td>
              <td className="p-2">{p.minStock}</td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  )
}
```

## Definition of Done

- `/products` menampilkan data dari database.
- Loading state terlihat saat data diproses.
- Tidak ada error rendering di browser.
